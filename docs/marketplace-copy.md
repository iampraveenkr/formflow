# Google Workspace Marketplace Copy (Draft)

## App Name
FormFlow

## Short description
Automate Google Forms submissions into actions like email, sheets, webhooks, docs, and tasks.

## Detailed description
FormFlow helps teams automate repetitive follow-up work after form submissions. Configure triggers, conditions, and actions in a no-code workflow builder. Runs are logged with step-level visibility, retries, and plan-aware controls.

### Key capabilities
- Form-triggered workflows
- Conditional routing (all/any mode)
- Action automation (email, webhook, sheets, docs, calendar, Slack)
- Run logs with retry diagnostics
- Workspace-level isolation and plan controls

## OAuth consent explanation
FormFlow requests only the scopes needed to execute selected actions. For example, Gmail scope is required to send email actions and Sheets scope is required to append rows.

## Install flow copy
Connect your Google account, choose permitted scopes, and verify your workspace settings. You can disconnect at any time from Integrations.

## Permission explanation UI copy
"FormFlow uses these permissions only to execute actions you configure. Data is scoped to your workspace and logged for auditability."
